## Summary

Applying Claude skills to communication, leadership, conflict resolution, and personal growth. Technical skills improve tasks; soft skills improve people throughput; personal growth skills improve long-term operator quality. The compounding effect often makes soft skills higher ROI than technical skills.

## Key Points

- Most developers stall from communication gaps, poor prioritization, low visibility - not lack of coding ability
- Claude skills train better behavior daily as embedded coaching (not one-off workshops)
- Technical skills improve tasks; soft skills improve people throughput; personal growth skills improve operator quality
- High-value skills: executive communication, conflict resolution, feedback delivery, leadership presence
- Developer career pack: `clear-status-update`, `show-impact`, `speak-like-senior`, `professional-boundaries`, `weekly-career-review`
- Skill = best for personal development; Agent = best for coaching programs

## Notes

Two developers can code equally well. One gets promoted because they communicate clearly, look reliable, influence decisions, show impact, and handle pressure professionally. Claude can help train that behavior daily.

Use Claude less as a coder and more as a **career operating system**.
